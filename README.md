🖥️Мобильная версия (Flutter + Dart)
-------

🖋️Технологии: Flutter + Dart

🖋️Мобильное приложение разработано как демонстрационная версия системы AIS Warehouse (Складской учёт). Оно предназначено для демонстрации возможностей мобильной адаптации складской системы

🖋️Приложение не подключено к серверу и работает в автономном режиме с фиктивными данными.

🖋️Основной функционал: Авторизация (демонстрационный режим), Просмотр списка товаров, Просмотр журнала операций, Просмотр статистики, Просмотр профиля пользователя

ИСТОЧНИК ДАННЫХ
----------------

Для демонстрации используются:

локальные фиктивные данные

модели в оперативной памяти

Серверная часть не реализована. Подключения к БД нет




❗ВНИМАЕНИЕ:❗
----------
Обращаю ваше внимание, что все персональные данные (имена, фамилии, email‑адреса, название компаний, адреса, контактные лица, ИНН и номера телефонов), представленные в таблицах и на скриншотах, носят исключительно вымышленный характер. Любое сходство с реальными людьми и компаниями является непреднамеренным и случайным
